cocoflow
========

Concurrency Control Flow
并发流程控制

### To build:

Prerequisites (Linux):

    * GCC 4.2 or newer
    * Python 2.6 or 2.7
    * GNU Make 3.81 or newer
    * (All of above is for libuv)
	
Linux:

    make
	
Prerequisites (Windows):

    * UnZip (You can extract dependencies by other tools)
    * Python 2.6 or 2.7
    * Visual Studio 2008 or 2010 or 2012
	
Windows:

    run vcbuild.bat
	
### To run the tests:

Linux:

    cd test/
	make all_test
	
Windows:

    cd vc\
	run test.bat
